|------|------|
| Project | Dự án tạm thời tạo ra sản phẩm |
| Operation | Hoạt động lặp lại |
| Stakeholder | Bên liên quan |
| Deliverable | Sản phẩm bàn giao |

| Risk | Rủi ro |
| Issue | Vấn đề đang xảy ra |
| Change Request | Yêu cầu thay đổi |
| Agile | Linh hoạt, thích ứng |

| Hybrid | Kết hợp |
| Incremental | Giao từng phần |
| Iterative | Lặp cải tiến |
| Value | Giá trị mang lại |
| Benefits | Lợi ích dài hạn |
| Tailoring | Tùy chỉnh phương pháp |
| KPI | Chỉ số đo hiệu suất |
