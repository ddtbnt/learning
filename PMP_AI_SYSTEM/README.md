# PMP_AI_SYSTEM

## Mục tiêu hệ thống

PMP_AI_SYSTEM là framework local AI dùng:

- Docker Desktop
- Ollama
- Qwen3.5

để:

- chuẩn hóa nội dung học PMP
- làm giàu kiến thức
- sinh câu hỏi ôn tập
- sinh đề thi thử
- xây web app nội bộ

## Vai trò từng module

| Thư mục | Chức năng |
|---------|-----------|
| 01_Setup | Cài Docker + Ollama + model |
| 02_Content_Standardization | Chuẩn hóa tài liệu học |
| 03_Question_Generation | Sinh câu hỏi và đề thi |
